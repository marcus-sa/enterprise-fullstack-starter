### Getting started

Assuming you've used the repository as a template, and that you have the project open in your favorite IDE.

#### Prerequisites
* A running [Kubernetes](https://github.com/kubernetes/kubernetes) cluster
* [Backyards](https://github.com/banzaicloud/backyards-cli#installation) components installed in your cluster
* [Skaffold](https://github.com/GoogleContainerTools/skaffold)
* [Yarn](https://github.com/yarnpkg/yarn)
